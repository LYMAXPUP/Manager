# Manager
demo - 资产管理系统

Tomcat本地部署，Vue + Servlet + jdbc

## 基础功能：
* 网页登陆、注册功能；
* 资产列表：资产(批量)导入、领用、报废、编辑；
* 我的申请：查看已领用资产，归还、报废、编辑；
* 人员配置：
  * 普通用户：编辑、重置密码、注销；
  * 超级用户：禁用/解禁用户
